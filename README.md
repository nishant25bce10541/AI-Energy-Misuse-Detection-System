# ⚡ Energy Misuse Detection using Machine Learning

## 📌 Overview

This project uses machine learning to detect energy misuse based on consumption patterns.

Energy misuse causes major losses to power companies. This project helps identify suspicious users by analyzing their energy usage.

---

## 🎯 Features

* Generates synthetic energy consumption data
* Detects abnormal usage patterns
* Uses machine learning models:

  * Logistic Regression
  * Random Forest
* Provides accuracy and confusion matrix
* Simple data visualization

---
